How

Sharing Spoon relies on a vast array of technical tools for functionality. This particular course didn't allow for backend, so everything you see is vanilla HTML/CSS/JS, with some API integration to make things run smoothly. I relied heavily on the functionality provided by the MealDB and EmailJS to create the site you see here. A huge thank-you to Professor Xin and Chris Ray for help along the way.
