Frequently Asked Questions

What is a Mutual Meal Plan?

A mutual meal plan is a planner which allows for several individuals to make and manage a rotating number of dishes, usually for the purpose of making cooking easier or more equitable.

Is There a Mobile App??

Unfortunately, this class only covers web-based development. Ideally, this website would be viewable from a mobile browser, but there may be some compatibility (and skill) issues with conversion to an app.
